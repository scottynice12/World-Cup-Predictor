# ⚽ World Cup Match Predictor

A Python/HTML hybrid web application that simulates World Cup football matches using a Poisson distribution-based prediction engine.

## 🎯 Features

- **Smart Predictions**: Uses team strength ratings (70-92) + random form factors
- **Poisson Distribution**: Realistic goal scoring simulation
- **20+ Teams**: Includes top international sides
- **Interactive UI**: Clean, responsive dark theme
- **Instant Results**: Click and predict in real-time
- **One-Click Reset**: Quick return to default matchup

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/world-cup-predictor.git
cd world-cup-predictor

# Run the application
