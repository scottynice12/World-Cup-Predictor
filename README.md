# ⚽ World Cup Match Predictor

A Python/HTML hybrid web application that simulates World Cup football matches using a Poisson distribution-based prediction engine with **48 international teams**.

## ✨ Features

- **🏆 48 Teams**: Complete World Cup roster with accurate strength ratings
- **🧠 Smart Predictions**: Poisson distribution + team strength + random form factors
- **🎨 Modern UI**: Glass-morphism design with responsive dark theme
- **⚡ Instant Results**: One-click predictions with winner announcement
- **🔄 Real-time Updates**: Team strength display updates on selection change
- **📱 Mobile Ready**: Fully responsive for all devices
- **🔌 One-Click Reset**: Quick return to default matchup
